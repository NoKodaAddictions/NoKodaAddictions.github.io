File Database Project - An application to store and request data


Intro

This is File Database. An application to store and request data. Just like Excel or Google Sheets,
you can store data in a file and request it. Now with version 1.7+, you can create multiple databases.

Every Download Button Will Have (I-M-R, M-R, or R)

I-M-R = Data is stored on the Internet, downloaded to memory and run on RAM
M-R = Data is stored on Memory and run on RAM
R = Data is stored and run on RAM
