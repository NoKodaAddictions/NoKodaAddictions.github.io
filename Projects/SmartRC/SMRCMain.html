SmartRC Project

Intro

SmartRC is a GUI based controller. It includes the ability to detect keypresses and update itself.
This program is currently in alpha stages, and it is a bodge. It will be redone after a physical
client prototype has been created.

Demo



The current GUI displays only flight data, command replies and 4 input buttons. (Fig 1)
The update screen changes depending on whether there is a client and/or a module update.
The user has the option to update the program now or later, but the update window will keep
popping up until it has been updated.

Future Changes

After a physical prototype has been made [either a Car or Airplane], the code will be rewritten.
The current script is a bodge and a proof of concept for the actual program and GUI.
The main part of the SmartRC program will be the launcher. It will be able to contain
multiple "interfaces". Along with installing "1st party interfaces", the client
will be able to host "3rd party interfaces". Most likely, the way to install these
interfaces would be through a "setup" or "linker" file. [idk why I named it that]
The linker file will include the name of the interface, the git repository, and some
other stuff.
