HeatSense Project 1.1 Concept

What Has Changed?

A case has been created to hold all the parts together.

PICTURE IS NOT TO SIZE

The case is 5in*4.5in The part that holds the Rasperry Pi 3 is 3.5in*2.5in. The second
comparments hold the battery, speaker and the 2 sensors [PIR + DHT22]. There is also
a lid created for the case [the holes on the edge]. The new system has been rewritten.
The program is now multi-threaded, and can be paired to an Android phone via bluetooth
and the HeatSense App.
