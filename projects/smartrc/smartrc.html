Intro

SmartRC is an application that uses the flask framework to manage and host other programs called "interfaces". As the name states, these interfaces have a remote control capability. However, any program with any purpose can be used.

Old [Legacy] Designs

The first version of SmartRC was built as a controller for remote control aircraft. Over time, that definition has changed to being able to host and manage other remote control apps.

As you can see, this version of SmartRC used the built-in tkinter framework. The version display on the bottom right corner of the client window shows the commit id. This is because the program checks for updates on the GitHub repository and uses 'git pull' to download and install it.

Note: Redesign was never finished

The next redesign of SmartRC used the PyQt5 Framework along with the PyQt5 Builder. This was a real pain since whenever I converted the .ui file into a .py file, the elements would be very small, making me go through every element and resize it.
