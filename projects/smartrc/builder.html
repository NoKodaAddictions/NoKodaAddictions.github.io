Intro

The SmartRC Builder is a python library to help you create and configure your application for the SmartRC manager. As of this time, the builder has thse modules: Comms, Linker and Project.

NOTE: smartrcB version 0.3 is still in development. Some module demos will not be shown

smartrcB versions before 0.3 are no longer supported. [GitHub Gists have also been deleted]
